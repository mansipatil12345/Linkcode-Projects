# Portfolio Card

A modern and responsive portfolio card built using **HTML** and **CSS**. This project showcases a clean personal profile card with a professional layout, smooth styling, and an attractive color scheme.

## ✨ Features

* Responsive portfolio card design
* Clean and modern UI
* Built using only HTML and CSS
* Font Awesome icons
* Google Fonts integration
* Hover effects and smooth transitions
* Beginner-friendly code structure

## 🛠️ Technologies Used

* HTML5
* CSS3
* Google Fonts
* Font Awesome
